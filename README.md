# Low resouce Translation
This project was created using this template [here](https://github.com/fastapi/full-stack-fastapi-template)
This github repo contains the full stack part of a larger project.
To view the live project you can go [here](https://low-resource-ranslation.gaelgilbello27.workers.dev/)

In the live project a user is given the ability to translate sentences from spanish to english. If you want to check the code for training and testing the model it is in a notebook [here](https://github.com/GaelGil/notebooks/blob/master/transformer/main.py)
